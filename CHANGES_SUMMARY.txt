┌───────────────────────────────────────────────────────────────────────┐
│ 1. SCENARIO FIELD FOR CHARACTERS                                     │
└───────────────────────────────────────────────────────────────────────┘

  Added a new "Scenario" field to Characters (not User Characters)
  
  GUI: Characters Tab
  ┌─────────────────────────────────────────────────────────────┐
  │ Scenario:  [You are in a fantasy tavern, ready to tell  ]  │
  │            [stories to adventurers...                    ]  │
  │            (Default situation/scenario for this character)  │
  └─────────────────────────────────────────────────────────────┘
  
  Use Case: Define the default context for your character
  Example: "You are a detective in a noir-style 1940s city"

┌───────────────────────────────────────────────────────────────────────┐
│ 2. EDIT FUNCTIONALITY - CHARACTERS                                    │
└───────────────────────────────────────────────────────────────────────┘

  Before: Could only add or delete characters
  Now:    Can edit existing characters
  
  Workflow:
  1. Select character from list
  2. Click [Edit Selected] ← NEW
  3. Modify fields in form
  4. Click [Update Selected] ← NEW
  5. Or [Clear Form] to cancel ← NEW
  
  All fields editable:
  ✓ Character Name
  ✓ Display Name
  ✓ Description
  ✓ Scenario ← NEW
  ✓ Avatar URL
  ✓ Avatar File

┌───────────────────────────────────────────────────────────────────────┐
│ 3. EDIT FUNCTIONALITY - USER CHARACTERS                               │
└───────────────────────────────────────────────────────────────────────┘

  Same edit functionality added to User Characters tab
  
  Workflow: Identical to Characters edit workflow
  Buttons: [Edit Selected] [Update Selected] [Clear Form]
  
  Note: User Characters do NOT have scenario field

┌───────────────────────────────────────────────────────────────────────┐
│ 4. !IMAGE BOT COMMAND                                                 │
└───────────────────────────────────────────────────────────────────────┘

  Update character avatars directly from Discord!
  
  Usage Option 1 - URL:
  ┌───────────────────────────────────────────────────────────────┐
  │ !image dashie https://i.imgur.com/cute-rainbow-dash.png      │
  └───────────────────────────────────────────────────────────────┘
  
  Usage Option 2 - Attachment:
  ┌───────────────────────────────────────────────────────────────┐
  │ !image dashie                                                 │
  │ [Attached: avatar.png]                                        │
  └───────────────────────────────────────────────────────────────┘
  
  What it does:
  ✓ Downloads the image
  ✓ Saves to character_avatars/ directory
  ✓ Updates character's avatar_url
  ✓ Updates character's avatar_file
  
  Supported formats: PNG, JPG, JPEG, GIF, WEBP

┌───────────────────────────────────────────────────────────────────────┐
│ 5. AI CONFIGURATION AUTO-SAVE                                         │
└───────────────────────────────────────────────────────────────────────┘

  AI Configuration Options (Presets tab) now save automatically
  
  Before: Settings were lost on restart
  Now:    Settings persist across sessions
  
  When you click "Save Configuration" in Configuration tab, it saves:
  ✓ Max Tokens / Context Length
  ✓ Response Length
  ✓ Temperature
  ✓ Top P
  ✓ Model Reasoning (enabled/disabled)
  ✓ Reasoning Level
  ✓ Presence Penalty
  ✓ Frequency Penalty
