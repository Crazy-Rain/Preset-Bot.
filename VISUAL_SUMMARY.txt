║ VALIDATION CHECKS                                                            ║
╚══════════════════════════════════════════════════════════════════════════════╝

  ✓ URL Format              → Must start with http:// or https://
  ✓ Server Accessible       → HTTP HEAD/GET request succeeds
  ✓ Image Type              → PNG, JPG, JPEG, GIF, or WEBP
  ✓ File Size               → Warn if > 2MB, reject if > 8MB
  ✓ Network Errors          → Timeout, connection failures
  ✓ Error Messages          → Clear, actionable feedback
